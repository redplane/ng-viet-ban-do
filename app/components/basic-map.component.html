Options
Properties Type Description
backgroundColor string Color background của mapDiv
center LatLng Center của Map lúc khởi động.
layer Layer hoặc [layer1,layer2,....] Khởi động Map với layer. Nếu layer là null thì lấy layer mặc định.
disableDoubleClickZoom boolean Cho phép double click zoom Map hay không. Mặc định là false
scrollWheel boolean Cho phép scroll chuột để zoom Map hay không. Mặc định là true
limitBounds LatLngBounds Giới hạn khung bao của map. Mặc định là null
mapTypeId MapTypeId Khởi động Map mapTypeId. Mặc định là ROADMAP.
minZoom number Mức zoom level nhỏ nhất của Map
maxZoom number Mức zoom level lớn nhất của Map
zoom number Mức zoom của Map lúc khởi động.
isMoveInsert boolean Nếu true thì khi drag Map sẽ tự động vẽ lại, Ngược lại dragend mới vẽ lại. Mặc định là true
extendTile number Mở rộng Map. Mặc định là 0.
zoomControl boolean Khởi động Map có control zoom, và zoom fit. Mặc định là true
zoomControlOptions Object
scaleControl boolean Khởi động Map có scale Control. Mặc định là true
scaleControlOptions Object
